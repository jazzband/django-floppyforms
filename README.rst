Installation
------------

* ``pip install -e git+git://github.com/brutasse/django-floppyforms.git#egg=floppyforms``
* Add ``floppyforms`` to your ``INSTALLED_APPS``

For extensive documentation see the ``docs`` folder or `read it on
readthedocs`_
